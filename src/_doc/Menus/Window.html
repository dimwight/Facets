This menu has the following items:

New
Opens a new content window on the currently active file; multiple windows on the same file are numbered sequentially.

All windows share the same page selection but can have different pane layouts.

Close
Closes the active window, and the active file if exposed in only one window.
Tile Windows
Rearranges all content windows to fill the current content area as equally as possible.
Scale to Desktop
When checked, content windows are resized with the application window; otherwise they remain the same size.

Below the menu items is a list of currently open windows, with the active window checked.
