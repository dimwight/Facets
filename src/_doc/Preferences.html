pdfInspect allows the user to set a range of preferences, which are stored as part of the application's session state.

Pane layout
Selection and arrangement of view panes can be defined in the Select Panes dialog.
Timeouts and error recovery
The Watch page of the Preferences dialog allows setting of application response to operation timeouts and failures
File preferences
The File page of the Preferences dialog allows setting of
  • the number of recent files to appear in the File menu
  • file load behaviour on application launch
