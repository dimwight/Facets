Language framework based on TreeCodeds.

Allowing application data and operations to be defined in a TreeCoded source tree, Eval provides a framework for simple, robust 'mini-languages' addressed to non-technical users.

Eval code

Whereas for most programming languages code is edited as plain text which is then parsed into a tree structure, the Eval tree is designed to be edited directly.

  • Application dialects are not expected to be Turing-complete nor programs to be elaborate: dialect types can encapsulate and hide any underlying complexity.
  • The DataNode coding of TreeCodeds is easy to represent graphically compared with say the complex typology of XML.

Eval types

While Eval can encode a tree of arbitrary TreeCodeds as defined by its interpreter, the core dialect defines generally useful types based on

  • Value representing immutable data as text, a number or a flag, or (non-recursively) a set of child Values.
  • Evaluable representing a generic operation returning a (possibly empty) Value.

Further types may be defined by dialects such as that implemented by treecode.form.

Java types

This package implements the Eval core with the following types:

Value
TreeCoded implementing Value.
Evaluable
TreeCoded implementing Evaluable.
EvalContext
TreeCodeContext that returns only Evaluables.
EvalTypes
TreeCodeTypes that can create their contents in a supplied EvalContext.
EvalCoded
Container Evaluable that can reference other Evaluables supplied by an EvalContext that it shares with children created using a supplied EvalTypes.

The following Evaluables provide general-purpose functionality:

Values
Evaluates to a Value summing its contained Evaluables and Values.
ValueOf
Simply evaluates the Evaluable referenced by its Label.
ValueMatching
Evaluates to varying Values to match evaluation of the Evaluable referenced by its Label.
IndexedValue
Evaluates to a Value matching an index set.
IfValue
Evaluates the Evaluable referenced by its Label, and returns a flag Value depending on the match with its child Value.
