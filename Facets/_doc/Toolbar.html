The toolbar has the following elements:

Page Number
An editable field with Next and Previous buttons, followed by the page total for the active content.
Text Size
Buttons increase/decrease the text size in all text displays, active when a text view pane is visible.
Page Render
Radio buttons selecting the render style, active when the Rendering view pane is visible.
  • Text Only renders text elements only
  • Text and Graphics renders all page elements
