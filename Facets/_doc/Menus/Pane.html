This menu has the following items:

Select Panes...
Launches the Select Panes dialog.
Layout
This sub-menu has two items:
  • Flip rotates the complete pane layout through a right-angle.
  • Reset restores the current user defaults as set in the Select Panes dialog.

Both items are also available in the dialog.

Maximise Active
Hides all view panes except the pane that currently has the focus; disabled for a single view pane.
Restore All
Shows all view panes; only enabled when a pane has been maximised.
