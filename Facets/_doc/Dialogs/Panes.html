This dialog is accessed from the Pane menu. It allows the user to

  • choose which view panes are displayed and how they are laid out
  • define a default pane layout and when this should be confirmed

Pane selection and arrangement

The available view panes are grouped as follows:

  • Pages, Document and Trailer panes under Structure
  • Rendering, Extracted and Stream panes under Page

Panes checked appear in the layout; panes in each group appear in sash panes unless the Show in Tabs checkbox for the group is checked. The Flip and Reset buttons duplicate the corresponding items in the Pane menu.

As settings are changed the display changes to match. Pressing OK confirms the layout; if the dialog is cancelled all changes are abandoned.

Once a layout has been confirmed for the content it is saved to session state and applied whenever this content is loaded.

Options

Pressing the Options>> button extends the dialog with two checkbox items:

Set as default
When this item is checked before confirming the layout, the layout will be applied whenever unknown content is loaded.
Always ask
When this item is checked, the dialog will be opened whenever unknown content is loaded.
