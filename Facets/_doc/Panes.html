The Select Panes dialog allows the user to choose which view panes are displayed and how they are laid out.

View panes are divided into two categories::

  • Structure panes display tree views of complete PDF content.
  • Page panes display text and graphic views of the active page.

Structure panes

Pages
Shows all page dictionaries in order (a flattened view of the pages dictionary).
Document
Shows the complete PDF object tree.
Trailer
Shows the PDF trailer which includes the pages dictionary tree.

Page panes

Rendering
Shows a rendering of the current page, either text only or both text and graphics.
Extracted
Shows the text content of the current page.
Stream
Shows the PostScript code of the current page.

Relationships between view panes and PDFBox architecture are detailed in pdfInspect and PDFBox.
